# ⚙️ Installation & Setup

## Clone the Repository

```bash
git clone https://github.com/dhruvgoe/Portfolio.git
```

## Navigate to the Project Directory

```bash
cd Portfolio
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

---

# 🏗️ Build for Production

```bash
npm run build
```
